HTML

  • HTMLstands for Hypertext Markup Language.
  • It is one of the principle languages used in web development.
  • There are various 'elements' that make up HTML which provide structure to web pages.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

Git

  • A branch is a copied version of code that can be edited in isolation with no disruption to the original/default.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • It's important to check that your local and remote branches are synced before pushing changes to a repo.
  • You can check this by typing 'git pull origin main' in your command line.
